# Summer Vibes Tunisia 2026 — Landing Page Easy2Book

Landing page promotionnelle pour Easy2Book — campagne été 2026.

## Stack

- **Next.js 14** (App Router)
- **Tailwind CSS 3**
- **TypeScript**

## Démarrage local

```bash
npm install
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000) → redirige vers `/summer-vibes`.

## Logo

Placer le fichier `logo.jpg` dans le dossier `/public/`.

## Déploiement

Compatible **Vercel** (recommandé) ou **Netlify**.

```bash
npm run build
```
